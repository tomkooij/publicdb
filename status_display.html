Station Status Display

The Status Display provides webpages that display summaries of data measured by stations.

The following packages and modules are included:

nagios
definitions that access nagios (vpn.hisparc.nl) to retrieve station status
urls
url resolvers
views
definitions which return pages with data for the HiSPARC stations
